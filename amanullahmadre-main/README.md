# Amanullah Madre — Portfolio

Personal portfolio website built with React, TypeScript, Vite, shadcn-ui, and Tailwind CSS.

## Getting started

Requires Node.js & npm installed.

```sh
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build

# Preview the production build locally
npm run preview
```

## Contact form setup

The contact form sends messages via EmailJS. Copy `.env.example` to `.env` and fill in your EmailJS Service ID, Template ID, and Public Key before building.

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion
- EmailJS
